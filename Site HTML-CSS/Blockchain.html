Qu'est ce qu'est une blockchain ?

Une blockchain est à la base de la crytomonnaie. Chaque opération effectuée avec une cryptomonnaie est stockée dans ce qui est appelée une blockchain.Et c'est ces blockchains qui rendent une cryptomonnaie plus sure. Une blockchain est une technologie de stockage et de transmissions, aussi appelé une base de données où sont enregistrés chaque opération effectuée par ses utilisateurs, toutes opérations est anonyme et infalsifiable. Elle n'a pas d'organe central de contrôle, c'est-à-dire qu'il n'y a pas de serveur unique, la blockchain est hébergée par ses utilisateurs.

Il existe de types de blockchain : une blockchain publique et une blockchain privée.

  • Une blockchain publique, signifie que chaque utilisateur peut lire cette base de données et en possédée une copie.
  • Une blockchain privée peut être consulté par ce qui le souhaite mais ne peut pas être modifié. Seul l'organisation peut modifier le protocole quand elle le souhaite.
A l'heure actuelle il existe plusieurs centaines de blockchain différentes.

Comment fonctionne une blockchain ?

  1. A effectue une transaction vers B,

  2. La transaction est regroupée à d'autres et forment un bloc,

  3. Le bloc est validé,

  4. Il est ajouté à la chaine de bloc(la blockchain), rendu visible par l'ensemble du réseau,

  5. B reçoit la transaction.
